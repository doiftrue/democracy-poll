# Widgets

1. Open **Democracy Poll → Settings** and enable **Widget**.
2. Open **Appearance → Widgets**.
3. Add the **Democracy Poll** widget to a widget area.
4. Select a specific poll, the last open poll, or a random active poll.
5. Choose whether the poll question should become the widget title.

On singular content, a poll attached through the post metabox takes precedence when that feature is enabled.

Block themes may expose classic widgets through a Legacy Widget block. If the active theme has no widget areas, use a shortcode block instead.
