# Design and appearance

Open **Democracy Poll → Design** to select a poll theme and configure controls without editing plugin files.

![The complete design screen in WordPress Admin.](/screenshots/admin-design.png){.doc-screenshot}

## Available controls

- poll theme and checkbox/radio appearance;
- maximum poll height and transition speed;
- result progress calculation and colors;
- progress-line height and animation;
- button preset, colors, hover colors, and additional button classes;
- AJAX loader;
- custom CSS appended after generated styles.

The progress bar can represent each answer relative to the winning answer or as a percentage of all votes.

## Custom CSS

Use **Custom/Additional CSS styles** for small site-specific adjustments. The generated CSS displayed at the bottom of the page helps identify selectors currently in use.

To replace the bundled theme entirely, select **No theme** and provide all required styles. Prefer a child theme or version-controlled site plugin for extensive customizations so changes remain reviewable.

See [Markup and CSS](/developer/markup-css) before replacing complete poll layouts.
