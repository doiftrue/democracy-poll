# Create your first poll

1. Open **Democracy Poll → Add Poll**.
2. Enter the poll question.
3. Add at least two answers. Drag answers to change their order.
4. Leave **Activate this poll** enabled.
5. Choose optional behavior such as custom answers, revoting, multiple answers, or a closing date.
6. Select **Add Poll**.
7. Copy the displayed shortcode, for example `[democracy id="12"]`, into a post or page.

![The complete add-poll screen in WordPress Admin.](/screenshots/admin-edit-poll.png){.doc-screenshot}

The poll ID appears in the polls list and in the shortcode shown on the edit screen. See [Display polls](/user-guide/displaying-polls) for every placement method.
