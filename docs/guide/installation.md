# Install the plugin

## From WordPress Admin

1. Open **Plugins → Add New Plugin**.
2. Search for **Democracy Poll**.
3. Select **Install Now**, then **Activate**.
4. Open **Democracy Poll → Add Poll**.

## From a ZIP file

1. Download the plugin ZIP from [WordPress.org](https://wordpress.org/plugins/democracy-poll/).
2. Open **Plugins → Add New Plugin → Upload Plugin**.
3. Upload the ZIP, install it, and activate it.

Deleting the plugin from WordPress Admin removes its options, polls, answers, and logs. Back up the database before uninstalling if you may need that data later.
